# 05 — Navigate a persistent weekly Client grid

**What to build:** A Member can organize and navigate a dense Client-by-date weekly workspace whose rows remain useful across weeks without ever hiding recorded work.

**Blocked by:** 04 — Manage Clients and derived first-use state.

**Status:** ready-for-agent

- [ ] My time opens the current Swedish-local Monday-to-Sunday week by default.
- [ ] Previous- and next-week navigation preserves the seven-day grid structure and follows Swedish-local dates.
- [ ] A Member can add an active Client as a personal standing row that remains present in later weeks.
- [ ] A Member can remove an unused standing row without deleting any Time Entry.
- [ ] Any Client with Time Entries in the displayed week appears even if its standing row was removed or the Client was archived.
- [ ] Standing-row preferences are private to the Member and do not create or alter Time Entries.
